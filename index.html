1. Discord OAuth Setup (Recommended to start with)

Since Instagram OAuth has account creation issues, let's get Discord working first:

  1. Go to [Discord Developer Portal](https://discord.com/developers/applications)
  2. Create a new application or use existing one
  3. Go to "OAuth2" ‚Üí "General"

2. Instagram OAuth Setup (When Meta account issues are resolved)

In your Instagram Basic Display app settings, set the redirect URI to:

https://ikarman4.github.io/oauth-callbacks/instagram/callback

3. Update Your Environment Configuration

Edit androidApp/src/main/assets/env.properties:

4. Test the OAuth Flow

Launch your Android app and try connecting Discord first. You'll be redirected here after authorization.

üîç How It Works

  1. User clicks "Connect Discord/Instagram" in your Android app
  2. App opens OAuth page in browser
  3. User authorizes your app
  4. Platform redirects to this GitHub Pages site
  5. Site displays the authorization code
  6. User copies the code and returns to the app
  7. App exchanges the code for an access token
